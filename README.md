## Deployment

To deploy this project run following command in the project folder

Clone the code repository
```bash
  https://github.com/robertoricci/Kidney-Disease.git
```

virtualenv venv
```bash
  python -m venv .venv
```

virtualenv activate
```bash
  source env/bin/activate
```

 Install 
```bash
  pip install -r requirements.txt
```

 Runn 
```bash
  flask run
```
